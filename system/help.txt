*Administrativos:*


**++off**
*Encerra a execução*

**++info**
*Exibe informações técnicas*

**++save**
*Salva as configurações atuais*


*Gerais:*


**++help / ++ajuda / ++h / ++aj**
*Exibe os comandos*

**++meeting / ++reunião / ++mt / ++rn**
*Cria uma reunião. Ex:
++meeting [Nome da reunião]*

**++remove_meeting / ++remover_reunião / ++rmt / ++rrn**
*Remove uma reunião. Ex:
++remove_meeting [Nome da reunião]*

**++start / ++iniciar / ++st / ++in**
*Inicia uma reunião. Ex:
++start [Nome da reunião]*

**++stop / ++parar / ++s / ++p**
*Para uma reunião*

**++add / ++adicionar / ++a**
*Adiciona um tópico em uma reunião. Ex:
++add [Nome da reunião] [Nome do tópico] [Tempo de duração do tópico em minutos]*

**++remove / ++remover / ++r**
*Remove um tópico de uma reunião. Ex:
++remove [Nome da reunião] [Nome do tópico]*

**++add_member / ++adicionar_membro / ++addm / ++am**
*Adiciona um membro na reunião para o controle de frequência. Ex:
++add_member [Nome da reunião] @<Menção ao usuário>*

**++add_members / ++adicionar_membros / ++addms / ++ams**
*Adiciona todos os membros do servidor na reunião para o controle de frequência. Ex:
++add_member [Nome da reunião]*

**++remove_member / ++remover_membro / ++removem / ++rm**
*Remove um membro da reunião. Ex:
++remove_member [Nome da reunião] @<Menção ao usuário>*

**++register_frequency / ++registrar_frequencia / ++rf / ++chamada**
*Faz uma chamada com os membros da reunião.*

**++show_frequency / ++mostrar_frequencia / ++mf / ++sf**
*Exibe a frequência dos membros na reunião. Ex:
++show_frequency [Nome da reunião]*

**++meetings / ++reunioes / ++mts / ++rns**
*Exibe as reuniões no servidor.*

**++members / ++membros / ++ms**
*Exibe os membros de uma reunião. Ex:
++members [Nome da reunião]*

**++topics / ++topicos / ++ts**
*Exibe os tópicos de uma reunião. Ex:
++topics [Nome da reunião]*
